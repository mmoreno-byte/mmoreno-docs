# Portfolio personal

Sitio web personal desarrollado con React y Vite. Mi primer proyecto profesional
con despliegue automático mediante GitHub Actions.

## Tecnologías

- **React 19** — Librería principal de UI
- **Vite** — Bundler y servidor de desarrollo
- **CSS3** — Estilos personalizados con animaciones
- **GitHub Actions** — CI/CD automático

## Qué hace

- Sección Hero con iconos de tecnologías
- Grid de proyectos con efecto hover
- Sección "Sobre mí"
- Footer con links a redes sociales

## Cómo ejecutarlo localmente

```bash
git clone https://github.com/mmoreno-byte/portfolio
cd portfolio
npm install
npm run dev
```

## Lo que aprendí

- Estructura de componentes en React
- Despliegue automático con GitHub Actions
- Diseño responsive con CSS Grid
